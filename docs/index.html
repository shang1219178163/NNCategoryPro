分类扩展，少写样板，多写业务

组件化核心模块：用 Category 扩展 Foundation / UIKit， 覆盖 Runtime、列表复用、防崩溃、日期与金额等常用能力。

iOS 12+ Objective-C CocoaPods · SPM MIT
Install 
pod 'NNCategoryPro'
dependencies: [
    .package(
        url: "https://github.com/shang1219178163/NNCategoryPro.git",
        from: "7.10.4"
    )
]

模块速览

按场景浏览常用 Category。输入关键词可即时筛选。

RT
Foundation

Runtime 与 Hook

遍历 ivar / 属性 / 方法 / 协议，便捷 NSCoding，方法交换与防护 Hook。

[obj enumerateIvars:^(Ivar v, NSString *name, id value) { NSLog(@"%@ = %@", name, value); }]; swizzleInstanceMethod(cls, @selector(a), @selector(b));
HO
集合

高阶函数

数组 / 字典的 map、filter、reduce 风格遍历，少写 for 循环样板。

NSArray *names = [list map:^id(User *u) { return u.name; }]; NSArray *valid = [list filter:^BOOL(id obj) { return obj != nil; }];
UI
UIKit

视图工厂

UILabel / UIButton / UITextField 等一键创建与链式配置，加速搭界面。

UILabel *lab = [UILabel createRect:frame]; UIButton *btn = [UIButton createRect:frame title:@"确认"]; [view addSubviewChain:lab];
CV
列表

Table & Collection

Cell / HeaderFooter 注册与复用封装，布局与空白页辅助方法。

[tableView registerClass:MyCell.class]; MyCell *cell = [tableView dequeueReusableCell:MyCell.class]; [collectionView registerClass:MyCell.class];
DT
时间

DateFormatter 缓存

线程字典缓存 Formatter，日期字符串与时间戳互转，避免重复创建。

NSString *s = [NSDateFormatter stringFromDate:date fmt:kDateFormat]; NSDate *d = [NSDateFormatter dateFromString:s fmt:kDateFormat];
CP
稳定

数组字典防崩溃

CashProtector 对 NSArray / NSDictionary 危险 API 做防护，降低线上 Crash。

// Hook __NSArrayI / __NSDictionaryM 等 // 越界、nil key/value 写入时安全降级 [NSObject openCashProtector];
¥
金额

金额数字动画

UILabel 金额滚动展示，配合数字格式化，适合资产与账单场景。

[label animationFromNumber:@(0) toNumber:@(1999.99) duration:1.2];
+
资源

图片 · 颜色 · 字符串

SDWebImage 封装、十六进制颜色、AES / Base64、Bundle 资源读取等。

[imageView sd_setImageWithURL:url]; UIColor *c = UIColorHexValue(0xF05138); NSString *enc = [plain AES128EncryptKey:key encodeType:@"1"];
